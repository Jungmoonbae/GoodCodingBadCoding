1. 중괄호 위치
 
   함수의 시작이나 블록의 시작점과 끝점을 알려주는 { }(중괄호)는 들여쓰기가 굉장히 중요 !

첫 번째 Style : 중괄호를 문장과 한줄에 쓰는 방법
<code>
 void main() {
  int i,jl
  for(i=2;ij<=9; i++) {
     for(j=1; j<=9; j++){
       printf("%d x %d = %2d", i,j,i*j); }
    printf("\n");}}
</code>
